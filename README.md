# 🚀 ShipTracker Pro - Advanced Shipping Management System

A stunning, modern shipping management application built with Next.js, React, and Tailwind CSS featuring a beautiful dark theme with gradient accents and comprehensive authentication system.

## ✨ Features

### 🔐 Authentication System
- **Secure Login/Register**: Beautiful authentication pages with form validation
- **Protected Routes**: Route guards to protect sensitive pages
- **Session Management**: Persistent login state with localStorage
- **Demo Credentials**: Easy testing with pre-configured admin account

### 📦 Shipping Management
- **Advanced Dashboard**: Responsive card layout with gradient backgrounds
- **Real-time Search**: Search by Shipment ID or Receiver Name
- **Smart Filtering**: Filter by status and date ranges with instant results
- **Detailed Tracking**: Individual shipment pages with complete history
- **Status Updates**: Simulate real-time status changes with notifications

### 🎨 Beautiful Dark UI
- **Modern Dark Theme**: Sleek black backgrounds with gradient accents
- **Glass Morphism**: Backdrop blur effects and translucent cards
- **Gradient Elements**: Beautiful blue-to-purple gradients throughout
- **Smooth Animations**: Hover effects and transitions
- **Responsive Design**: Perfect on all devices

## 🛠️ Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: JavaScript/JSX
- **Styling**: Tailwind CSS with custom dark theme
- **UI Components**: shadcn/ui with custom styling
- **Icons**: Lucide React
- **Authentication**: Context API with localStorage
- **State Management**: React Hooks

## 🚀 Quick Start

1. **Install dependencies**
   \`\`\`bash
   npm install
   \`\`\`

2. **Run development server**
   \`\`\`bash
   npm run dev
   \`\`\`

3. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

4. **Login with demo credentials**
   - Email: `admin@shiptracker.com`
   - Password: `admin123`
